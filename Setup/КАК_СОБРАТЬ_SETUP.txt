═══════════════════════════════════════════════════════════════
  ПОЛНЫЙ ГАЙД: как сделать установщик (setup) для «Кондитерская»
═══════════════════════════════════════════════════════════════

Файл скрипта:  Setup\ConfectionerySetup.iss
Готовый setup: Setup\Output\ConfectionerySetup_v1.0.0.exe


─── ЧТО НУЖНО УСТАНОВИТЬ НА СВОЙ ПК ───────────────────────────

1) Visual Studio (уже есть) — для сборки проекта
2) Inno Setup 6 — для создания .exe установщика
   Скачать: https://jrsoftware.org/isdl.php
   Выбрать: «Inno Setup 6.x.x» → установить
   При установке можно отметить «Inno Setup Preprocessor» (уже входит)


─── ШАГ 1. Собрать программу в Release ─────────────────────

1. Откройте решение в Visual Studio:
   D:\OOP\Project_OOP\Lab_06_OOP\Lab_06_OOP.sln  (или .csproj)

2. Меню: Сборка → Configuration Manager

3. Установите:
   Configuration:  Release
   Platform:       Any CPU

4. Закройте запущенное приложение «Кондитерская», если оно открыто

5. Меню: Сборка → Пересобрать решение

6. Проверьте, что появился файл:
   D:\OOP\Project_OOP\Lab_06_OOP\bin\Release\Lab_06_OOP.exe

   В той же папке должны быть .dll и Lab_06_OOP.exe.config


─── ШАГ 2. (По желанию) Фото товаров в установщик ────────────

Если в админке вы добавляли фото, они лежат здесь:
   %AppData%\Confectionery\Images\

Инсталлятор скопирует их на компьютер пользователя.
Если папки нет — setup всё равно соберётся, просто без картинок.


─── ШАГ 3. Открыть скрипт в Inno Setup ───────────────────────

На экране «Welcome» (как у вас на скриншоте):

  ✓ Выберите:  «Open an existing script file»
  ✗ НЕ используйте «Script Wizard» — скрипт уже готов!

1. Нажмите «More files...»
2. Укажите файл:
   D:\OOP\Project_OOP\Setup\ConfectionerySetup.iss
3. Нажмите OK

Откроется редактор с текстом скрипта.


─── ШАГ 4. Скомпилировать установщик ─────────────────────────

1. В Inno Setup: меню Build → Compile
   (или клавиша Ctrl+F9)

2. Внизу в окне «Compiler Output» должно быть:
   «Successful compile» / «Compile completed successfully»

3. Если ошибка:
   «#error Сначала соберите проект...»
   → вернитесь к ШАГ 1 (Release не собран)

4. Готовый файл:
   D:\OOP\Project_OOP\Setup\Output\ConfectionerySetup_v1.0.0.exe

Этот .exe можно отправить другу / сдать на курсовую.


─── ШАГ 5. Проверка установщика ──────────────────────────────

1. Запустите ConfectionerySetup_v1.0.0.exe
2. Пройдите мастер установки
3. Запустите «Кондитерская» из меню Пуск
4. Вход админа: admin@mail.ru / admin


─── ЧТО ДОЛЖНО БЫТЬ НА КОМПЬЮТЕРЕ, КУДА СТАВЯТ ─────────────

| Компонент              | Зачем                          |
|------------------------|---------------------------------|
| Windows 7 SP1+         | Минимум для .NET 4.7.2         |
| .NET Framework 4.7.2+  | Проверяется установщиком       |
| SQL Server Express     | База данных (.\SQLEXPRESS)     |

Без SQL Express программа установится, но при запуске
может не подключиться к базе.

Строка подключения в App.config:
   Data Source=.\SQLEXPRESS;Initial Catalog=ConfectioneryDb;...

На другом ПК должен быть установлен SQL Express с таким же
именем экземпляра (SQLEXPRESS) или нужно изменить App.config
перед сборкой Release.


─── ЧАСТЫЕ ОШИБКИ ───────────────────────────────────────────

Ошибка при Compile Inno Setup:
  «File not found» / #error Release
  → Соберите проект в Release (Шаг 1)

Ошибка при сборке в Visual Studio:
  «файл используется другим процессом»
  → Закройте запущенное приложение

После установки программа не запускается:
  → Установите .NET 4.7.2 и SQL Server Express

Фото не видны после установки:
  → Перед Compile setup добавьте фото в админке
  → Или скопируйте папку Images в AppData вручную


─── КРАТКАЯ ШПАРГАЛКА ───────────────────────────────────────

Visual Studio:  Release → Пересобрать
Inno Setup:     Открыть ConfectionerySetup.iss → Ctrl+F9
Результат:      Setup\Output\ConfectionerySetup_v1.0.0.exe
